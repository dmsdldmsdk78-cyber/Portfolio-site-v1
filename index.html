퍼피노트 UI를 피그마로 설계하고 화면 흐름을 정리했습니다.

홈, 마이페이지, 다이어리, 지도, 케어 화면까지 전체 구조를 먼저 잡고,
사용자가 자연스럽게 이동할 수 있도록 화면 간 흐름과 구성을 시각화했습니다.

Project 01

Will you be my Valentine 💌

네 / 아니오 버튼의 서로 다른 반응을 중심으로 구성한 인터랙티브 카드 프로젝트입니다. 버튼 클릭 이벤트, 동적 위치 변경, 화면 상태 전환을 구현하며 JavaScript 기반 UI 제어 방식을 익혔습니다.

Project 02

계산 기능을 넘어 사용자가 누르는 재미를 느끼는 UI를 목표로 만들었습니다.

Project 03

Typing Rain Game 🎬

이와이 슌지 영화에서 착안한 산성비 스타일 타이핑 게임입니다. 떨어지는 단어를 입력해 제거하는 구조로, 게임 루프와 점수 반영, 상태 변화에 따른 화면 업데이트를 구현했습니다.

프로젝트 바로가기 Game Logic DOM

Project 04

Photo Booth 📸

사용자의 카메라를 활용해 사진을 촬영하고, 다양한 스티커를 배치해 꾸민 뒤 완성된 이미지를 저장할 수 있는 웹 포토부스입니다.
브라우저의 MediaDevices API로 실시간 카메라 영상을 가져오고, DOM 기반 드래그 배치를 통해 스티커를 자유롭게 배치할 수 있도록 구현했습니다.
완성된 화면은 Canvas로 합성하여 하나의 이미지로 다운로드할 수 있습니다.

프로젝트 바로가기 Camera API Canvas Image Compose
